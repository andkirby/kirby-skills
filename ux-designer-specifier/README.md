# UX Designer Specifier

A skill that writes UX specs and ASCII wireframe schemas for frontend components and surfaces in any project.

## What it does

Produces structured UX composition specs, state tables, and standalone ASCII wireframe diagrams in markdown — removing guesswork for implementation. Not a code generator; defines the UX contract (what, how composed, what states exist) without owning runtime code.
